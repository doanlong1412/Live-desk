# 💜 LiveDesk — Live2D Anime Waifu cho Home Assistant

Một nhân vật anime Live2D sống trên dashboard Home Assistant của bạn — chào bạn bằng tên, phản ứng với sensor thời gian thực, cảnh báo khói hay chuyển động, và nói chuyện bằng TTS.

**Ghim nhân vật** để cô ấy nổi trên toàn bộ dashboard, luôn hiện diện dù bạn đang ở view nào.

## ✨ Tính năng nổi bật
🌐 Full bilingual support — English and Vietnamese, switchable live from the editor; greetings, sensor reactions, alerts, device tooltips, and all UI text adapt instantly

🏠 Bilingual device tooltips — hover descriptions for all 17 entity domains now fully translated
